# Binary Tree Art

Generate a .jpg file of a random binary tree.

## Requirement 
- python3.11 +
- PIL(Pillow) 9.3.0 +

## Execute

```
> python main.py [option]
```
**Noticed: Images will be saved under the /img folder**

Argparse Option:
    `- num`: numbers of the image.
    `- layer`: layers of the binary tree.
